Business Chosen

Life Care Diagnostics – Diagnostic Center

Project Overview

This project was created as part of Future Interns Prompt Engineering Task 1.

The objective was to design a structured prompt system that generates professional and patient-friendly website copy for a diagnostic center. The generated content includes homepage copy, services page content, and persuasive call-to-action sections.

Tools Used

- ChatGPT
- GitHub
- Lovable

Prompt Framework

The prompt system was designed to:

- Generate homepage copy with a clear value proposition
- Create service descriptions for a diagnostic center
- Write persuasive CTA sections
- Maintain a professional, trustworthy, and patient-friendly tone
- Produce website-ready content

Prompt Logic

The prompts were structured to:
